Solidity Smart Contract as an Example

Lifecycle

Write Contract Codes in Solidity

Developer writes code with Solidity, a domain specific language for EVM smart contract.

Compile

Compile the smart contract codes to bytecode with the EVM toolchains.

Deploy

Deploy the bytecodes to the blockchain (Ethereum or Arbitrum) by sending a transaction and pay the gas fee.

Execute

Users trigger a smart contract function with a transaction, which also requires paying the gas fee based on the invovled exectuions.

Solidity — The EVM Standard

Strengths

  • Native EVM integration

  • Mature ecosystem & tooling

  • Widely audited patterns

  • Battle-tested security

Limitations

  • Learn a new language

  • Limited standard library

  • Expensive gas cost
